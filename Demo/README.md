## 🎯 **Demo Scenarios**

**Try It Now:**
```bash

  -H "Content-Type: application/json" \
  -d '{
    "scenario": "federal_reserve_tightening",
    "horizon_months": 12,
    "confidence_level": 0.95
  }'
```

**What You'll See:**
- Real-time inflation predictions
- Economic driver analysis

- Interactive charts and visualizations
